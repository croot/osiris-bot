# Requirements #

```
- Python >= 2.7 (Not 3.x)
- Subversion
- Linux/*BSD/*NIX system :)
```


# Installation #

```
# download
svn co http://osiris-bot.googlecode.com/svn/trunk osiris

# configure
cd osiris/settings
cp demo_config.py config.py
nano config.py
cd ..

# launch
python osiris.py &
```
